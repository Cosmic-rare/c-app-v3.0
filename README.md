# Chat-application(c-app/Tani)

いい感じに作成したチャットアプリです。
DiscordやSlackをまねながら、自分たちで使う専用のチャットアプリを作っています。

# DEMO

まだUIしか作れていませんが、[Vercel](https://c-app-v3-0.vercel.app/)にデプロイしておきました。

# How to Use

```bash

$ git clone https://github.com/Cosmic-rare/c-app-v3.0.git

$ cd c-app-v3.0

$ npm install

$ npm run start
```
